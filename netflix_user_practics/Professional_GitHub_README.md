## 🚀 Project Overview

This project analyzes a Netflix User Behavior dataset using PostgreSQL and demonstrates real-world business analytics skills.

The repository covers:

- Customer Behavior Analysis
- Revenue Analytics
- Churn Analysis
- User Segmentation
- Engagement Metrics
- Window Functions
- CTEs
- Advanced SQL Reporting
- Interview-Focused SQL Problems

# 🎯 Business Objectives

✔ Identify high-value customers
